## Summary
Step-by-step repotting technique and the critical rule about how much to up-size — too-large a jump causes waterlogging and rot, a major risk for citrus. Defines correct execution when increasing pot size is actually chosen.

## Key Findings
- **Size up only one step at a time: +2–5 inches diameter** (one or two pot sizes). **Never jump to a much larger pot** — excess mix stays saturated and causes root rot.
- **Best timing: late winter/early spring** before the growth flush, so the tree recovers fast during the active season.
- Technique: ease the root ball out, **tease/shave circling outer roots, set the crown/graft above the soil line**, backfill with fresh fast-draining mix, water in.
- For trees at target size, **root-prune and return to the SAME pot** instead of up-sizing.
- Match canopy pruning to root disturbance to keep root-to-canopy balance (Topic 7).

## Relevance to 19" × 14" Container
Defines correct technique and a key warning. If ever moving up from the 19" pot, go only one size larger — a too-big container would drown the shallow-rooted tree (the same perched-water problem from Topic 1, amplified). But for the stated 10-year goal the 19"×14" is likely the terminal pot, so the operative procedure is **root-prune + refresh in place each late winter**, crown kept above the mix. Up-size only if the tree clearly needs more vigor/size.
