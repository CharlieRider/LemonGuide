## Summary
Description of UC's C-35 citrange, a semi-dwarfing rootstock — the middle option between full dwarfing (Flying Dragon) and standard vigor (Carrizo). Relevant as the fallback if Flying Dragon is unavailable and a less extreme size reduction is acceptable.

## Key Findings
- **C-35** = Ruby Blood orange × trifoliate orange, **released by UC in 1987**.
- Produces a **semi-dwarf tree, ~83% of Carrizo size (≈15–25% smaller)**, roughly 10–15 ft in-ground.
- **Good cold hardiness/frost tolerance** from trifoliate parentage; trifoliate lineage also brings Phytophthora tolerance.
- Less extreme dwarfing than Flying Dragon (which is ~30–50% of normal size), so a C-35 Lisbon in a pot will still be vigorous and need active size control.

## Relevance to 19" × 14" Container
Secondary option. C-35's ~17% size reduction is far less than Flying Dragon's 50–70%, so for the 14" pot it is a compromise — acceptable only with aggressive, sustained pruning of the vigorous Lisbon scion. Prefer Flying Dragon for the container goal; consider C-35 only if FD stock is unavailable and you accept more pruning labor.
