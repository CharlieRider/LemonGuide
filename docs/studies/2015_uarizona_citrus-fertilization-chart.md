## Summary
Desert-Southwest extension fertilization chart giving an N schedule split across the season for a hot, low-humidity climate. A useful hot-climate cross-check on the cooler-coastal UC IPM schedule, since Brentwood summers run hot even with marine moderation.

## Key Findings
- Mature citrus target on the order of **~1 lb actual N per year** for a full-size tree, **split into ~3 applications** (late winter/pre-bloom, late spring, and summer) to match the long Southwest growing season.
- Splitting N reduces salt-spike and leaching loss versus a single heavy application — the same logic that drives "weekly weakly" in containers.
- Alkaline desert soils make **Fe/Zn/Mn deficiency routine**, corrected with chelates/foliar micronutrients — mirroring the container high-pH problem.

## Relevance to 19" × 14" Container
Climate-direction relevant (hot season length), but rates are for in-ground full-size trees and must be cut sharply and further subdivided for a 14" pot. The value here is the **multi-split scheduling principle** for a long, hot growing season and the confirmation that alkaline-soil micronutrient lockout is the norm, not the exception.
